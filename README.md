# 🎩 不思議の国のアリス風 謝恩会招待状サイト

クラシック・ビンテージスタイルの「不思議の国のアリス」をテーマにした謝恩会招待状用の特設ウェブサイトです。

## ✨ デザインコンセプト

**黒・白・赤を基調としたクラシックな配色**
- 古い絵本や白黒写真のようなレトロな雰囲気
- ビンテージペーパーの質感
- トランプのモチーフ（♠ ♥ ♣ ♦）
- 白ウサギと赤い薔薇のアクセント

## 🎨 デザイン要素

### カラーパレット
- **メインカラー**: 黒（#000000）、白（#FFFFFF）
- **アクセントカラー**: ダークレッド（#8B0000）
- **背景**: ビンテージペーパー（#F5F5DC）

### モチーフ
- 🐇 白ウサギ
- ♠ ♥ ♣ ♦ トランプカード
- 🌹 赤い薔薇
- ⏰ 懐中時計
- 🫖 ティーセット

### タイポグラフィ
- **英語**: Playfair Display、Cinzel Decorative（装飾的なセリフ体）
- **日本語**: Noto Serif JP（明朝体）

## 🚀 使い方

### 1. ローカルで表示

```bash
# そのままブラウザで開く
open index.html

# またはローカルサーバーで起動
python3 -m http.server 8000
# その後 http://localhost:8000 にアクセス
```

### 2. ブラウザで確認

`index.html` をブラウザにドラッグ&ドロップするだけでも動作します。

## ⚙️ カスタマイズ方法

### イベント情報の変更

`index.html` の以下の部分を編集してください：

```html
<!-- 日時の変更（55行目付近） -->
<p class="detail-text">2025年3月20日（木）<br>18:00 開宴</p>

<!-- 場所の変更（62行目付近） -->
<p class="detail-text">ワンダーランドホール<br>〒123-4567<br>東京都○○区○○1-2-3</p>

<!-- 連絡先の変更（95行目付近） -->
<p class="contact-info">
    または<br>
    TEL: 090-XXXX-XXXX<br>
    Email: example@example.com
</p>
```

### メッセージ内容の変更

`index.html` の74行目付近のメッセージボックス内のテキストを変更できます。

### 色の変更

`styles.css` で以下の色を変更できます：

```css
/* アクセントカラーを変更したい場合 */
#8B0000  /* ダークレッド → お好みの色に */
```

### RSVPボタンの連携

`script.js` の162-168行目あたりで、実際のフォームやメールリンクに変更できます。
現在はアラートボックスが表示される設定です。

## 🎭 特徴

### ビジュアルデザイン
- **ビンテージペーパー**: 古い紙の質感を再現
- **装飾的なボーダー**: 四隅の装飾フレーム
- **二重枠デザイン**: 各カードの重厚感のある枠線
- **セピア調**: レトロな雰囲気の配色

### インタラクティブ機能
- **クリックエフェクト**: トランプマーク（♠♥♣♦）が舞う
- **ホバーアニメーション**: カードが浮き上がる
- **スクロールアニメーション**: 要素がゆっくり現れる
- **控えめなキラキラ**: 定期的にマークが輝く
- **隠しコマンド**: ↑↑↓↓ で白ウサギの緊急モード！

### レスポンシブ対応
- スマートフォン
- タブレット
- デスクトップPC

## 🎮 隠し機能

キーボードで以下を入力してみてください：

```
↑ ↑ ↓ ↓
```

白ウサギが「遅刻する！」と叫びます🐇⏰
